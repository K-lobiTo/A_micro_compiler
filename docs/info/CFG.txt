Guide of the modified Context-free grammar for the language

Prog: program
id: identifier
sc: semicolon (;)
Stmts: statements
Stmt: statement
Expr: expression
Val: value
plus: plus (+)
comment: minus (-)
assign: designate a value
read: read a value from terminal
write: print a value in terminal
begin: begin of program
end: end of program
